DESCRIPTION

General

/

The master root/home page.

Administrative Backend

All administrative functions other than /admin/$iface/login require a valid login. If you are not logged in, you will be redirected to the login page before being able to continue.

You can specify the username and password query parameters to automatically log in with the request. This is required when using a non-interactive interface (e.g. xml).

/admin

UNIMPLEMENTED The main administrative home page.

/admin/$iface

The main administrative page for $iface. UNIMPLEMENTED FOR XML

/admin/$iface/commit

Commit any outstanding changes to the Solr index.

/admin/$iface/docs

Show the menu of avaibale POD files.

/admin/$iface/docs/$docfile

Display $docfile which is the name of a POD file (minus the .pod suffix) in the docs directory.

/admin/$iface/finish/$contributor/$directory

Removes the BagIt directory from the uploads area of the server.

/admin/$iface/ingest?cc=$contributor [file=$file_upload]

Upload the BagIt archive $file_upload to the server into the upload subdirectory for $contributor.

/admin/$iface/login

Log in to use the administrator interface.

/admin/$iface/logout

Log out of an existing session.
