Les Ressources parlementaires historiques canadiennes constituent une base de données interrogeable gratuitement qui contient des publications parlementaires numérisées depuis la 1re session de la 1re législature (1867) jusqu’au début de leur diffusion sur sencanada.ca et noscommunes.ca.

Des explications sur le fonctionnement de la base de données se trouvent sur la page Aide à la recherche.

Ce site Web est le fruit d’une collaboration entre la Bibliothèque du Parlement et le Réseau canadien de documentation pour la recherche.
